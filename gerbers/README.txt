Always use the latest revision PCB.

"-gerbers-jlc.zip" is the PCB with JLCJLCJLC on the B Silkscreen layer for unique ID during board production and proper options for JLCPCB.  Be sure to select "specify a location" when ordering from JLCPCB.

"-gerbers-pcbway.zip" are the gerbers exported with the proper options for pcbway

Recommended board values:
	PCB Thickness: 1.6mm
	Copper Weight 1oz
	Castellated Holes - No
	Tented Vias - Yes
	Surface Finish - Personal preference
